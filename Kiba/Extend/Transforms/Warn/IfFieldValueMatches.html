Overview

rubocop:todo Layout/LineLength Prints single warning to STDOUT if the value of the given field matches the given value in rubocop:enable Layout/LineLength any rows

rubocop:todo Layout/LineLength Useful if you have skipped mapping/setting up transforms for certain values in an initial/staging rubocop:enable Layout/LineLength rubocop:todo Layout/LineLength data set, but need to ensure you will notice if later/production data includes new values rubocop:enable Layout/LineLength that need attention.

This transform warns if Utils::FieldValueMatcher finds a match.

def process(row)
  return row unless single_warnings.empty?

  result = matcher.call(row)
  return row unless result

  msg = "One or more rows has #{field} value matching #{match}"
  add_single_warning(msg)
  row
end
