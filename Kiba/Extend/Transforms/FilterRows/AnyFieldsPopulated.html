Overview

Keep or reject rows based on whether any of the given fields is populated. Blank strings and nils count as not populated.

Examples

Source data:

{a: 'a', b: 'b', c: 'c' },
{a: 'a', b: 'b', c: '' },
{a: '', b: nil, c: 'c' },
{a: '', b: 'b', c: 'c' },
{a: '', b: nil, c: nil },

Used in pipeline as:

transform FilterRows::AnyFieldsPopulated, action: :keep, fields: %i[a b]

Resulting data:

{a: 'a', b: 'b', c: 'c' },
{a: 'a', b: 'b', c: '' },
{a: '', b: 'b', c: 'c' }

Used in pipeline as:

transform FilterRows::AnyFieldsPopulated, action: :keep, fields: :all

Resulting data:

{a: 'a', b: 'b', c: 'c' },
{a: 'a', b: 'b', c: '' },
{a: '', b: nil, c: 'c' },
{a: '', b: 'b', c: 'c' }

Used in pipeline as:

transform FilterRows::AnyFieldsPopulated, action: :reject, fields: %i[a b]

Resulting data:

{a: '', b: nil, c: 'c' },
{a: '', b: nil, c: nil }

def process(row)
  finalize_fields(row) unless fields_set

  case action
  when :keep
    return row if any_populated?(row)
  when :reject
    return row unless any_populated?(row)
  end
end
